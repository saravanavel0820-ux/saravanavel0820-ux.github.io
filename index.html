Renewable Energy Monitoring System for domestic and Microgrids

Project Overview

Our project provides a comprehensive solution for managing local green power generation, specifically targeted toward hybrid domestic settings and decentralized community microgrids.

By interfacing physical power sensors (Voltage, Current, Battery banks) with an automated relay system, this architecture balances safety, self-consumption optimization, and cost savings.

Key Operations

  • Real-Time Analytics: Live status checking of household generation, net-metering balance updates, and deep-cycle storage reserves.
  • Dynamic Load Automation: Smart relay modules capable of dropping non-essential loads (like climate control units) when battery state drops.
  • Integrated Edge AI: local LLM assistant contextualized with live microgrid status registers for handling power shedding requests.
